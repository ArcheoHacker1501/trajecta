# Contributing to Trajecta

Thanks for your interest in contributing. We welcome bug reports, feature requests, and pull requests.

## How to Contribute
1. Fork the repository and create a new branch.
2. Keep changes focused and clearly scoped.
3. Add or update documentation if behavior changes.
4. Run the build before opening a PR.

## Reporting Issues
Please include:
- Operating system and version
- GPU model and CUDA version (if applicable)
- GDAL version
- Steps to reproduce
- Expected vs actual behavior

## Coding Style
- C++17
- Prefer clear, readable code over micro-optimizations
- Add comments only where the logic is non-obvious
